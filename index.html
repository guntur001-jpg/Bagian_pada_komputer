BAGIAN PADA KOMPUTER/PC
DAN PENGENALAN NYA

APA ITU KOMPUTER/PC?

Komputer atau PC (Personal Computer) adalah perangkat elektronik yang digunakan untuk mengolah data sesuai perintah dari pengguna. Komputer bisa menjalankan berbagai tugas seperti mengetik dokumen, browsing internet, bermain game, menonton video, membuat desain, coding, dan banyak lagi.

Fungsi Komputer:

  • Mengolah data menjadi informasi
  • Menyimpan dan mengambil informasi
  • Berkomunikasi dengan perangkat lain (internet, printer, dll)
  • Menjalankan perintah pengguna dengan cepat dan efisien

Komponen Pada Komputer/PC:

Komponen komputer terbagi menjadi 2 yaitu:

1. Hardware (Perangkat Keras)

Hardware adalah komponen fisik dari komputer yang bisa dilihat dan disentuh langsung. Tanpa hardware, komputer tidak akan berfungsi.

2. Software (Perangkat Lunak)

Software adalah program atau sistem berisi perintah untuk menjalankan fungsi tertentu di dalam komputer. Jika hardware adalah fisik, maka software adalah digital.

BAGIAN-BAGIAN HARDWARE

A) CPU

CPU (Central Processing Unit) adalah otak komputer yang bertugas mengolah data dan menjalankan perintah dari software.

  • Mengontrol jalannya program
  • Memproses data
  • Mengambil data dari RAM
  • Mengirim hasil ke perangkat lain

B) Monitor

Monitor menampilkan gambar, teks, video, dan informasi visual. Jenisnya meliputi CRT, LCD, LED, dan OLED.

C) Keyboard

Keyboard digunakan untuk mengetik atau memberi perintah. Jenisnya QWERTY, AZERTY, dan DVORAK.

D) Printer

Printer mencetak dokumen atau gambar ke media seperti kertas. Jenisnya Inkjet, Laser, Dot Matrix, dan Thermal.

E) Speaker

Speaker mengubah sinyal audio digital menjadi suara yang dapat didengar.

F) Mouse

Mouse digunakan untuk menggerakkan kursor dan memberikan perintah pada komputer.

BAGIAN-BAGIAN SOFTWARE

1. Software Sistem

Contoh: Windows, Linux, macOS, Driver, Antivirus.

2. Software Aplikasi

Contoh: Microsoft Word, Google Chrome, WhatsApp, Canva.

3. Software Pemrograman

Contoh: Visual Studio Code, Python, Java, C++.

4. Middleware

Contoh: API Gateway, Database Middleware.

Kesimpulan:
Software adalah otak dari komputer yang tidak bisa dilihat wujudnya, tetapi membuat komputer berguna.
